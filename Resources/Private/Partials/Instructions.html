Collapsible, read-only view of the editor-maintained agent instructions that inform the agent. "Always" instructions are injected into the system prompt; "on-demand" instructions are loaded by the agent via the GetInstruction tool when relevant. Editing happens via the tx_agent_instruction records in the List module (restricted to the permitted backend group). The body is rich text, rendered via f:transform.html (resolves t3:// links) piped through f:sanitize.html (XSS-safe in backend context). Renders nothing when there are no instructions.
