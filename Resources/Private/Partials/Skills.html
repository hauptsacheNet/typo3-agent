Collapsible, read-only view of the editor-maintained agent skills that inform the agent. "Always" skills are injected into the system prompt; "on-demand" skills are loaded by the agent via the GetSkill tool when relevant. Editing happens via the tx_agent_skill records in the List module (restricted to the permitted backend group). The body is rich text, rendered with f:format.html. Renders nothing when there are no skills.
